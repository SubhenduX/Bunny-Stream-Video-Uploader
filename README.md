# Bunny Stream Video Uploader

A WordPress plugin to upload videos to Bunny Stream via direct file upload or URL.

## Description

The Bunny Stream Video Uploader plugin allows you to upload videos directly to Bunny Stream from your WordPress dashboard. You can either upload video files directly or provide a URL to fetch the video.

## Installation

1. Download the plugin files and upload them to your WordPress site's `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin settings page via 'Bunny Stream Video Uploader' in the WordPress admin menu.
4. Enter your Bunny Stream API credentials and save the settings.

## Usage

1. Go to 'Bunny Stream Video Uploader' in the WordPress admin menu.
2. Enter the title of your video.
3. Either upload a video file or provide a URL to fetch the video.
4. Click 'Upload Video' to start the upload process.

## Settings

- **API Key**: Your Bunny Stream API key.
- **Library ID**: Your Bunny Stream Library ID.
